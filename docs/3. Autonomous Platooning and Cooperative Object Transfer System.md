# 개요
- 다중 이동 로봇이 Leader-Follower 구조로 협력 이동하며, 객체를 전달하거나 운반하는 협업 시스템을 구축이 목표
- 모바일 매니퓰레이터가 물체를 조작한 후, 후속 이동 로봇이 플래투닝 방식으로 따라오며 객체 전달 및 운반을 분담하는 구조로 구현
- 단일 로봇이 모든 역할을 담당하는 방식보다 역할 분담이 가능한 협력 구조를 형성하고, 전체 시스템의 확장성과 효율성을 높이기
- 다중 로봇 제어의 주기 안전성과 통합성을 고려하여 C++기반으로 구현
***
# 환경 설정
- **운영체제 :** Ubuntu 22.04
- **ROS 버전 :** ROS2 Humble
- **개발 언어 :** C++, ament_cmake
- **자율 이동 및 경로 계획 :** Nav2, odom, tf, RGB-D 센서 정보 활용
- **다중 로봇 환경 :** 로봇간 namespace 분리, ROS2 네트워크 설정 고려
- **Node 구현 :** 추종 제어, 리더 상태 송신, 협업 상태 관리
***
# 활용 알고리즘
- **Leader-Follower 기반 플래투닝 제어**
- 리더 로봇의 위치, 방향, 속도 정보 바탕으로 팔로워 로봇이 일정 거리와 정렬 상태를 유지하며 추종하도록 제어
- 기본 추종 제어는 거리 오차, 방향 오차를 이용한 PID 제어, 기하학 기반 추종 제어 방식으로 구현
- 자율 이동 구간에서 Nav2 경로 계획 및 장애물 회피 기능 활용, 리더 추종 필요 구간에서 플래투닝 제어 로직이 우선 적용
- 협업 객체 전달 과정에서 작업 완료 신호 수신히 팔로워 로봇이 지정 위치에 정렬, 적제/전달 단계로 이어지게 상태 기반 제어 구성
- C++ 기반 구현을 통해 다중 로봇 간 주기 제어 loop를 안정적으로 유지, 통신 지연 및 제어 지연 최소화를 목표
***
