# 3D Perception-Based Mobile Manipulator System

# 개요
본 프로젝트는 RGB-D 카메라 기반 3차원 객체 인식, 객체 좌표 기반 매니퓰레이터 정밀 제어, 플래투닝 기반 협업 운반 기능을 ROS2 Humble 환경에서 C++ 기반 패키지로 구현하는 것을 목표로 한다.

---
# 전체 시스템 구성
1. depth_perception
	- RGB-D 카메라 기반 객체 인식
	- CSRT 기반 객체 추적
	- Depth 및 RGB 기반 근거리 거리 보정
	- 객체의 3차원 좌표 추정

2. mp_control
	- perception 시스템에서 publish한 객체 좌표 subscribe
	- tf2 기반 좌표계 변환
	- MoveIt2 기반 매니퓰레이터 목표 pose 생성
	- Pick & Place 시퀀스 수행

3. po_delivery
	- Leader-Follower 기반 플래투닝 제어
	- 후속 이동 로봇의 객체 운반 및 협업 이동
	- 협업 상태 관리 및 안전 정지
---
# 실행 순서
1. RGB-D 카메라 및 perception 노드 실행
2. 객체 추적 및 3차원 좌표 publish
3. 매니퓰레이터 제어 노드 실행
4. 객체 접근 및 Pick & Place 수행
5. 플래투닝 기반 후속 로봇 운반 시스템 실행
6. 전체 통합 실험 및 결과 분석
