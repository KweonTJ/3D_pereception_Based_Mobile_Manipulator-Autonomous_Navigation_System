# 개요
- 3차원 비전 기반 객체 인지 시스템으로부터 전달 받은 객체의 3차원 공간 좌표를 이용해 매니퓰레이터가 목표 위치로 정밀하게 접근하고 객체를 조작하는 제어 시스템 구현이 목표
- 기존 고정 위치 기반 반복 동작이 아닌, 인식 결과에 따라 실시간으로 목표 좌표를 설정하고 해당 위치까지 로봇팔이 이동하도록 설계
- 이를 통해 작업 대상의 위치가 일부 변하더라도 인지 기반 유연한 조작이 가능, 최종적으로 Pick&Place 작업을 수행할 수 있는 정밀 제어 시스템 구축
- 전체 시스템 통합성, 제어 안정성을 고려하여 C++ 기반으로 구현
***
# 환경 설정
- **운영체제 :** Ubuntu 22.04
- **ROS 버전 :** ROS2 Humble
- **개발 언어 :** C++, ament_cmake
- **하드웨어** : Open Manipulator X
- **매니퓰레이터 제어** : rclcpp, tf2_ros, geometry_msgs, sensor_msgs, trajectory_msgs. motion planning을 위해 MoveIt2 API 활용
- 좌표 처리, 제어 노드를 모두 C++로 통일해 perception 시스템과 연동 지연 최소화 및 반복 제어 시 안정적 동작 확보 방향으로 설계
***
# 알고리즘
- **Inverse Kinematics**
- **Motion Planning**
- 3차원 비전 시스템에서 전달된 객체 좌표를 엔드 이펙터 목표 위치로 설정하고, 해당 위치에 도달하기 위한 관절 각도, 경로 계산
- MoveIt2의 planning interface를 활용해 목표 pose에 대한 경로 생성, 필요시 Catesian path planning을 이용해 보다 정밀한 접근 경로 구성
- 작업 동작은 접근 위치, grasp 위치, 상승 위치, 전달 위치, place 위치로 나눠 단계적 제어
- 좌표 오차를 줄이기 우해 tf2를 이용한 좌표계 변환, 그리퍼 기준 오프셋 보정, 작업 직전 위치 재확인 등의 보정 절차 포함
- 전체 제어 흐름은 C++ 기반 상태 머신, 시퀀스 제어 로직으로 구성, perception 입력부터 조작 완료까지 일관된 제어 구조

# 패키지 생성
### mp_control
- ament_cmake 기반 구성.
- **주요 구성 요소**
	- 객체 좌표 subscribe 노드
	- 좌표계 변환 노드
	- 목표 pose 생성 노드
	- MoveIt2 기반 motion planning 및 execution 노드
	- Pick & Place 시퀀스 제어 노드
	- 작업 상태 publish 노드
- **입력** : 3차원 비전 기반 객체 인지 시스템에서 publish한 객체 좌표 토픽. 로봇 베이스, 매니퓰레이터 기준 좌표계로 변환하여 사용
- **출력** : 관절 trajectory 명령, 작업 진행 상태, 조작 성공 여부 등의 정보로 구성
