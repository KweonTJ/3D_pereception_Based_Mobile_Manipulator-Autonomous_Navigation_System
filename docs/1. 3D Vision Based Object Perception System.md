# 개요
- RGB-D 카메라를 활용하여 작업 환경 내 객체를 인식하고, 해당 객체의 위치를 3차원 공간 좌표로 추정하는 시스템 구축을 목표로 한다.
- 모바일 매니퓰레이터가 물체를 안정적으로 조작하기 위해 실제 작업 공간 기준의 3차원 위치 정보가 필요하다. 이를 위해 RGB 영상과 Depth 영상을 함께 활요하여 객체를 검출하고, 객체의 중심과 목표 지점의 3차원 좌표를 산출하는 인지 시스템을 구현한다.
- RGB-D 데이터 처리 과정에서 발생할 수 있는 메모리 병목과 실시간 처리 문제를 고려하여 C++ 기반으로 구현하며, 추후 매니퓰레이터 정밀 제어 프로젝트에 입력될 객체 클래스 정보와 3차원 좌표 정보를 안정적으로 제공하는 것을 목적으로 한다.

# 알고리즘 및 제어 방식
## CSRT 알고리즘
- OpenCV에서 제공하는 고정밀 객체 추적 알고리즘.
- 상관 필더 기반 추적 기법을 개선한 방식. KCF 보다 정확도를 높이기 위해 채널 신뢰도와 공간 신뢰도를 동시에 고려.
- 입력 영상에서 초기 Bounding Box를 기준으로 객체의 특징을 학습한 뒤, 이후 프레임에서 해당 객체의 위치를 추적. 해당 과정에서 HOG, Color Names등 다앙한 feature 채널 활용.
- **Channel Reliability** : 각 feature채널의 중요도를 동적으로 평가하며, 신뢰도가 높은 채널에서 더 큰 가중치를 부여. 이를 통해 환경 변화에 강건한 추적 성능 확보
- **Spatial Reliability** : 객체 영역 내에서 실제 의미 있는 부분과 배경을 구분하여, 추적시 배경 영향 최소화. 마스크 기반으로 유효 영역만 반영하여 drift 문제를 감소.
- 프레임 각 객체 이동을 주파수 영역에서 빠르게 계산하는 Correlation Filter 특성 덕분에 실시간 처리 가능하며, 높은 정확도 유지
- CSRT 기반 2D 객체 위치를 안정적으로 추적하고, 바운딩 박스 영역을 ROI로 설정, 깊이 정보 추출

## IBVS
- 카메라 영상 정보를 이용해 로봇 제어 Visual Servoing 제어 방식
- 카메라 영상에서 얻은 특징점, 물체 중심 좌표, Bounding Box 중심, 이미지 오차 등을 이용하여 Manipulator의 움직임을 제어
	- 입력 : 이미지 상의 특징점 위치, 물체 중심 좌표, 픽셀 오차
	- 출력 : 로봇 속도, 자세 보정, end-effector 이동량
	- 목적 : 이미장의 목표 위치와 현재 위치의 오차 줄이기
- 이미지 평면상의 오차를 직접 줄이면서 로봇 제어
